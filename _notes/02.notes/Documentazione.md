# Documentazione

###  Documentazione Principale
- **Getting Started & Installazione**: guida passo-passo per Docker, Kubernetes (Helm), e deployment cloud
- **Ingestion & Connectors**: come configurare alimentatori per Snowflake, BigQuery, Kafka, dbt, Airflow, S3, ecc.
- **UI & Interazione**: ricerca, commenti, classificazione, ownership, piattaforme di analisi
- **Governance & Compliance**: classificazione dati, privacy tags, policy, audit, RBAC
- **Advanced**: personalizzazione ingestion, metriche, reporting, archiviazione export/import

###  Specifiche & Architettura
- **Open Metadata Specification (OMS)**: https://openmetadatastandards.org/
  Schema unificato, struttura dei metadati, contratti di API e linee guida per estensioni.
- **API Reference**: https://docs.open-metadata.org/v1.12.x/api-reference/main-concepts/metadata-standard/apis
  OpenAPI/Swagger completo per tutte le REST e GraphQL API (autenticazione, payload, esempi).
