# Data Insights

Data Insights in [[OpenMetadata]] are platform-level usage and health metrics that help data platform teams track adoption, engagement, and governance maturity. OpenMetadata tracks metrics such as:

- Which tables are most queried?
- What percentage of tables have descriptions?
- What percentage of tables have owners?
- Which datasets have the most open quality issues?
- How has data coverage improved over time?

These insights help teams prioritize documentation efforts, report on governance maturity, and identify areas needing attention.
