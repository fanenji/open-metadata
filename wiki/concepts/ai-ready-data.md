# AI-Ready Data

**AI-Ready Data** refers to data that has been enriched with enough semantic context (metadata) that an autonomous agent can interpret, query, and use without human intervention.

While "human-ready" data focuses on cleanliness and modeling for BI, "AI-ready" data requires:
- **Shared Definitions**: Clear, unambiguous meanings for all metrics and dimensions.
- **Lineage and Context**: Knowledge of where data originated and how it relates to other entities.

- **Machine-Readable Metadata**: Metadata that is accessible via protocols like **MCP** so that agents can programmatically discover and manipulate it.
