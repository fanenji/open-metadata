# Data Quality

Data quality in OpenMetadata refers to the ability to define and run tests on data to ensure its reliability and trustworthiness. It is a core feature of the platform.

## Purpose

Data quality tests help organizations maintain high standards for their data, detect anomalies, and prevent issues from propagating downstream.

## Integration

Data quality tests are defined and executed by the [[Ingestion Framework]], and the results are displayed in the [[OpenMetadata]] UI alongside the data assets.
