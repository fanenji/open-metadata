# Open Data Contract Standard (ODCS)

The **Open Data Contract Standard (ODCS)** represents an effort to standardize how data contracts are defined and shared across different platforms, preventing vendor lock-in and enabling a "best-of-breed" tool selection strategy.

### Key Features in OpenMetadata 1.12
- **Interoperability**: Support for ODCS 3.1 allows organizations to move contracts between OpenMetadata and other compatible tools.
- **Standardization**: By embracing ODCS, OpenMetadata ensures that data contracts remain a portable and foundational element of the modern data stack.
