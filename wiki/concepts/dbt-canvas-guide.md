# dbt Canvas Guide

A visual, interactive resource format for presenting comprehensive dbt knowledge. Created by [[Taylor Brownlow]] at [[Count.co]], the dbt canvas guide synthesizes documentation, community knowledge, and expert insights into a single visual interface. The guide is designed to serve both beginners and experienced dbt practitioners.

The canvas format differs from traditional blog posts by providing an interactive, map-like navigation experience rather than linear text. The guide covers dbt's purpose, key concepts, and reference materials in a visually organized manner.
