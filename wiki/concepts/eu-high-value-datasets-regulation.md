# EU High-Value Datasets Regulation

The EU High-Value Datasets (HVD) regulation (Implementing Regulation 2023/138) establishes a list of specific datasets whose reuse generates significant benefits for society, environment, and economy. It defines mandatory publication and reuse requirements for six thematic categories.

## Key Requirements

- **Machine-readable formats**: Data must be in open, machine-readable formats (at least 3-star on the 5-star model)
- **API + Bulk Download**: Dual access method required for most datasets (API for programmatic access, bulk download for batch processing)
- **Licensing**: CC BY 4.0 or CC0 (or equivalent open licenses)
- **Service Quality**: Performance, capacity, and availability criteria must be published for APIs
- **Contact Point**: A designated organizational unit must provide API feedback
- **Metadata**: DCAT-AP_IT for non-spatial data, RNDT for spatial data
- **Retroactive Application**: Rules apply to existing datasets published before June 9, 2024

## Six HVD Categories

1. **Geospatial**: Administrative units, geographic names, addresses, buildings, cadastral parcels, reference parcels, agricultural parcels
2. **Earth Observation/Environment**: 26 INSPIRE themes including hydrography, protected sites, elevation, geology, land cover, etc.
3. **Meteorological**: Weather observations, climate data, alerts, radar, NWP models
4. **Statistical**: Industrial production, CPI, national accounts, population, poverty, etc.
5. **Business/Company Ownership**: Company basic info, financial documents
6. **Mobility**: Transport networks

## Implementation Timeline

- **June 9, 2024**: Full application date for all HVD datasets
- **February 9, 2025**: Maximum extension for public administrations requesting 2-year exemption
