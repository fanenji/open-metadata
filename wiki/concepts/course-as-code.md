# Course-as-Code

**Course-as-Code** is the pattern of delivering an entire educational curriculum — lecture slides, notebooks, assignments, datasets, and syllabi — as a version-controlled, publicly accessible repository (typically on GitHub). This approach enables reproducibility, collaborative improvement, and transparent versioning of course materials.

The [[geog-414-spatial-data-management]] course by [[giswqs]] is a prime example: the full University of Tennessee curriculum is maintained as a GitHub repository, allowing students and self-learners to clone, run, and contribute to the materials. This pattern aligns with the wiki's existing coverage of open geospatial education resources like [[modern-gis-workshop]].
