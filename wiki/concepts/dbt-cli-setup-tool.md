# dbt CLI Setup Tool

A lightweight CLI tool built by [[BlaBlaCar]] to standardize and accelerate dbt project and profile creation across 50+ users. With a few inputs, it creates the complete project structure and profile configuration, eliminating repetitive manual setup. Part of the [[dbt-developer-experience]] and [[dbt-migration-strategy]] tooling.
