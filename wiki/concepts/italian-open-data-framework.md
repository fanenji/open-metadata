# Italian Open Data Framework

The Italian legal and technical framework for open data, based on EU Directive 2019/1024 and transposed via Legislative Decree 36/2006. Key components include:

- **AgID Open Data Guidelines (LG-OD)**: Technical rules for data opening and reuse, adopted with Determination 183/2023
- **Piano Triennale per l'Informatica nella PA**: Three-year plan for digital transformation, includes HVD implementation actions
- **DCAT-AP_IT**: Italian metadata standard for data catalogs
- **RNDT**: National Registry of Territorial Data for spatial metadata
- **Interoperability Guidelines (LG-INT)**: Technical standards for API implementation
- **API Security Guidelines (LG-SIC)**: Security standards for API interoperability

The framework mandates open-by-default, machine-readable formats, API access, and CC BY 4.0 or equivalent licensing for all public sector data.
