# Total Data Assets Widget

The **Total Data Assets Widget** is a UI component on the [[my-data-page]] that displays the trend of total data assets over the last 14 days. It shows counts for each asset type, including:

- Tables
- Dashboards
- Databases
- Database Schemas
- Pipelines
- Topics
- ML Models
- Charts
- Other data asset types

## Usage

Users can quickly identify the most populated data asset types in their organization. For example, an organization may have more Tables than any other data asset type.
