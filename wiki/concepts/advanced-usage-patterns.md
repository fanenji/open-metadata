# Advanced Usage Patterns

Advanced Usage Patterns covers the more sophisticated ways to use OpenMetadata's data quality test definitions beyond basic single-test execution. This includes chunk-based validation, publishing results, and best practices for production deployments.

## Topics Covered

- **Chunk-Based Validation**: Processing large datasets in chunks to manage memory and performance
- **Publishing & Best Practices**: How to publish test results to OpenMetadata and follow recommended patterns
- **Custom Tests**: Creating user-defined test definitions beyond the 25 built-in ones
- **Test Library**: Organizing and reusing test definitions across projects
