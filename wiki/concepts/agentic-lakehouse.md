# Agentic Lakehouse

The Agentic Lakehouse is Dremio's strategic vision for enabling AI agents to interact with data lakehouses using natural language. The [[dremio-mcp-server]] is positioned as a key enabler of this vision.

## Key Characteristics

- **Natural Language Interface:** AI agents and chat clients can query and explore data without SQL expertise.
- **MCP Protocol:** Built on the [[model-context-protocol]] standard for standardized AI-tool interaction.
- **Dual Deployment:** Available as both self-managed OSS and cloud-hosted managed service.

## Relevance

While the Agentic Lakehouse provides strategic context for the MCP server's existence, it is not directly actionable for current architecture decisions. The production readiness of the MCP server depends on deployment model and support obligations, not on the vision itself.
