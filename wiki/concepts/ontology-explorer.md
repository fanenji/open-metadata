# Ontology Explorer

The **Ontology Explorer** is an interactive, visual tool within OpenMetadata used to navigate, build, and govern business ontologies. It bridges the gap between abstract business definitions and concrete data implementations by mapping how glossary terms relate to each other and to the physical data assets that implement them.

## Key Features

- **View Modes**: Provides various perspectives for exploration, including **Overview**, **Hierarchy**, **Related**, and **Cross-Glossary**.
- **Data Traceability**: Enables users to trace business terms directly to the physical data assets that implement them—such as tables, dashboards, and pipelines—including their lineage and quality scores.
- **Visual Governance**: Facilitates "Active Governance" by allowing data stewards to create new term relationships directly on the map without writing queries.
