# Image Segmentation (Geospatial)

Image segmentation for geospatial data involves pixel-level classification of satellite and aerial imagery to identify and delineate features such as buildings, water bodies, wetlands, and solar panels. This is a core capability of geospatial AI systems.

Tools like [[GeoAI]] integrate with [[PyTorch Segmentation Models]] to provide:
- Automatic feature extraction from satellite and aerial imagery
- Specialized segmentation algorithms optimized for remote sensing data
- Streamlined workflows for segmenting specific feature types
- Export capabilities to standard geospatial formats (GeoJSON, Shapefile, GeoPackage, GeoParquet)
