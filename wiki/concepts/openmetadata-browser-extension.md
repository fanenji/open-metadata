# OpenMetadata Browser Extension

The OpenMetadata Browser Extension is a peripheral tool that provides quick access to OpenMetadata features directly from the browser. It is listed in the official user guide as a resource for data users to enhance their productivity.

## Purpose

The browser extension allows data users to:

- Quickly search and access data assets without navigating to the full OpenMetadata UI.
- View metadata, tags, and descriptions for data assets referenced in web pages or applications.
- Potentially trigger actions like following assets or viewing version history from the browser context.

## Availability

The extension is mentioned in the OpenMetadata v1.12.x user guide. Specific installation instructions and supported browsers are detailed in the official documentation.

## Relationship to Other Features

- Complements the [[my-data-page]] by providing an alternative access point for data discovery.
- Supports the [[data-user-perspective]] by reducing friction in accessing metadata.
- Works alongside [[personal-access-token]] for authentication if needed.

## Note

This feature is documented as a convenience tool for data users. Its exact capabilities and configuration requirements should be verified against the official OpenMetadata documentation.
