# HVD Implementation Actions (#hvd1-#hvd10)

The AgID operational guide defines 10 concrete actions for implementing the HVD regulation:

- **#hvd1**: Publish data in machine-readable formats (LG-OD Requirements 2 and 3)
- **#hvd2**: Provide API and bulk download access (LG-OD Requirements 7, 27, 28)
- **#hvd3**: For INSPIRE data, follow service quality regulations (EC 976/2009)
- **#hvd4**: For non-INSPIRE data, apply similar quality criteria and publish conditions
- **#hvd5**: Document non-INSPIRE APIs using OpenAPI 3.0 or WSDL 1.1
- **#hvd6**: Document INSPIRE APIs using Get Service Metadata/GetCapabilities
- **#hvd7**: Designate a contact point matching the metadata contact information
- **#hvd8**: Apply regulation retroactively to existing datasets
- **#hvd9**: Apply CC BY 4.0 or CC0 licensing (LG-OD Requirements 20 and 21)
- **#hvd10**: Apply licensing requirements to existing datasets as well
