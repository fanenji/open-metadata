# Environment-based Configuration

The transition from static, hardcoded, or `.properties`-based configuration to a standardized `.env` approach. This facilitates better separation of secrets and environment-specific parameters (e.g., Dev vs. Prod) and is implemented using libraries like `python-dotenv`.
