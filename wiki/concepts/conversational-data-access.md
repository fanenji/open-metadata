# Conversational Data Access

Conversational Data Access is a future trend predicted by [[Ritam Mukherjee]] where analysts run complex queries in the data lakehouse using plain English, powered by AI agents. This extends the [[text2sql-patterns]] concept by adding agentic capabilities for multi-step queries and data exploration.

## Current Status

While basic Text2SQL is available, conversational data access with multi-turn reasoning and complex analytical queries is still emerging. The article identifies this as a key future shift that will democratize data access for business analysts.
