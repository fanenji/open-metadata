# HVD Meteorological Category

The meteorological category of High-Value Datasets includes four datasets:

1. **Weather Observation Data**: Measured data from meteorological stations
2. **Climate Data**: Validated climate observations
3. **Weather Alerts**: Meteorological warnings and alerts
4. **Radar Data**: Weather radar observations
5. **NWP Model Data**: Numerical Weather Prediction model data (API-only access)

NWP model data is the only HVD dataset that must be made available exclusively via API (no bulk download requirement). Historical versions should be provided where possible.
