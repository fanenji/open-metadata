# Activity Feed Widget

The **Activity Feed Widget** is a core UI component on the [[my-data-page]] that displays all activities related to data assets the user owns, follows, or is mentioned in. It is the primary collaboration hub within the landing page.

## Display Sections

- **All** — Shows all activities related to owned, followed, or mentioned data assets.
- **@Mentions** — Filters feeds to show only those where the user is mentioned.
- **Tasks** — Displays open tasks created by or assigned to the user.

## Relationship to Pluggable Panels

The Activity Feed Widget is a concrete implementation of a [[pluggable-panels|pluggable panel]] that can be configured as part of the [[persona-and-landing-page-customization]] system.
