# My Data Widget

The **My Data Widget** is a UI component on the [[my-data-page]] that displays all data assets owned by the current user. It provides a quick glance at the user's owned assets and directs users to the Explore page to claim unowned assets.

## Usage

- Displays a list of all data assets (Tables, Dashboards, Databases, etc.) owned by the user.
- If the user or their team does not own any data assets, the widget encourages claiming assets from the Explore page.
