# Collate AI

**Collate AI** is a suite of generative AI and agentic features designed to automate the tedious aspects of metadata management and enhance data discovery.

## Core Components
- **AskCollate**: A conversational AI interface (available via web, Slack, and MS Teams) that allows users to query the metadata graph using natural language.
- **AI Studio**: A platform for creating and customizing enterprise-ready AI agents with specific personas, tools, and prompts.
- **AutoPilot Agents**: A collection of specialized agents designed for specific metadata tasks:
    - **Documentation Agent**: Automatically generates descriptions for tables and columns.
    - **Tiering Agent**: Assesses and tiers assets based on business criticality.
    - **Data Quality Agent**: Generates and manages data quality tests.
    - **Classification Agent**: Detects and tags sensitive data using AI recognizers.
- **AI SDK**: A developer toolkit for building custom AI applications powered by Collate's semantic metadata.

This suite represents the evolution of the metadata catalog from a passive repository to an active, **agentic** participant in the data ecosystem.
