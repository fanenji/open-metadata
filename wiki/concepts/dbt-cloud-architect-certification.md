# dbt Certified Architect Certification

The dbt Certified Architect is an official certification from dbt Labs for professionals who can architect, configure, and manage dbt Cloud deployments at enterprise scale. The certification exam tests knowledge across five technical domains.

## Exam Domains

1. **Connections** — Setting up data platform connections (Snowflake, Databricks, BigQuery, Redshift) and Git integration.
2. **Project Configuration & Environments** — Configuring dev, staging, and production environments; managing Git branching strategies.
3. **Security & Monitoring** — SSO, RBAC, licenses, permissions, and webhook-based automation.
4. **Governance & Discovery** — dbt Mesh for cross-project dependencies and dbt Catalog for documentation and discovery.
5. **Exam Preparation** — Advanced setup workshop, study guide, and pro tips.

## Learning Path

The official learning path (published on learn.getdbt.com) consists of six milestones, with required courses and elective resources. The path is currently in **beta** and is actively being updated based on learner feedback.

## Relevance

This certification represents the enterprise administration and architecture track for dbt, complementing the wiki's existing coverage of dbt technical patterns (contracts, testing, versioning). It introduces dbt Cloud as a managed platform with distinct features from open-source dbt.
