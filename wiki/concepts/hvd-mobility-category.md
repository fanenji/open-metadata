# HVD Mobility Category

The mobility category of High-Value Datasets includes one dataset:

1. **Transport Networks**: Road, rail, water, and air transport network data

This dataset must comply with INSPIRE data models for transport networks. Data owners include infrastructure managers and transport authorities. Publication requires machine-readable formats, API access, bulk download, and CC BY 4.0 licensing.
