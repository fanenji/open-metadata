# QGIS Plugin for AI

The QGIS plugin for AI enables users to run AI-powered geospatial workflows directly within the [[QGIS]] desktop environment without writing code. Developed as part of the [[GeoAI]] ecosystem, this plugin democratizes access to geospatial AI for non-programmers and GIS professionals.

The plugin allows users to perform tasks such as image segmentation, classification, and object detection through a graphical interface, abstracting the complexity of the underlying machine learning models and data processing pipelines.
