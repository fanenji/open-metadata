# Image Classification (Geospatial)

Image classification for geospatial data involves assigning land cover or land use categories to satellite and aerial imagery. This is a fundamental task in remote sensing and environmental monitoring.

Key capabilities include:
- Pre-trained models for land cover and land use classification
- Transfer learning utilities for fine-tuning models with custom data
- Multi-temporal classification support for change detection
- Accuracy assessment and validation tools
