# Key Performance Indicators (KPI) Widget

The **Key Performance Indicators (KPI) Widget** is a UI component on the [[my-data-page]] that displays governance metrics related to data asset coverage. It is a Data Insights component that provides ready information about:

- **Ownership Coverage** — Percentage of data assets that have an assigned owner.
- **Description Coverage** — Percentage of data assets that have a description.
- **Tiering Coverage** — Percentage of data assets that have a tier classification.

## Access

By default, the KPI Widget is accessible to Admins only. The documentation notes that other users can also be given access, but the exact mechanism (role assignment, policy configuration, or persona customization) is not specified in this source. See [[roles-and-policies]] and [[persona-and-landing-page-customization]] for related access control mechanisms.

## Relationship to Data Insights

The KPI Widget is part of the Data Insights feature. For troubleshooting KPI display issues, see [[data-insights-application-troubleshooting]].
