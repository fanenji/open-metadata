# Announcing OpenMetadata 1.12

This source announces the release of OpenMetadata 1.12, highlighting key features such as the **AI SDK** for programmatic access to the metadata knowledge graph, a new **Data Quality Test Library** for standardized SQL-based testing, and a **Kubernetes Orchestrator** that removes the dependency on Apache Airflow.

The release also emphasizes support for open standards like **ODCS 3.1** for data contracts and **OpenLineage** for lineage metadata ingestion. Additionally, it introduces semantic search capabilities using vector embeddings in **OpenSearch**.
