# Explore the Lineage View | Official Documentation

This source is the official OpenMetadata v1.12.x documentation page for the Lineage View. It describes the end-to-end lineage traceability UI for tables, columns, dashboards, and pipelines. The page covers the mechanics of exploring lineage (search, expand, click edges), the information displayed (SQL queries, pipeline info, column lineage), and the five-layer exploration framework: Column, Observability, Service, Domain, and Data Product layers.

Key topics include:
- Edge information (SQL query, source, target, description)
- Quick preview of data asset details (owner, tier, tags, schema, quality metrics)
- Lineage Config for upstream/downstream nodes (max 3)
- Pipeline-to-database and dashboard-to-database lineage
- The five lineage layers for multi-dimensional exploration

This source is the primary reference for the [[lineage-layers]] concept and the detailed UI behavior of the [[data-lineage]] feature.
