# Guida operativa sulle serie di dati di elevato valore (HVD) - Versione 1.0

## Overview

This document is the official operational guide published by the Agenzia per l'Italia Digitale (AgID) in December 2023. It provides guidance for Italian public administrations on implementing EU Implementing Regulation 2023/138, which establishes a list of specific High-Value Datasets (HVD) and their publication and reuse modalities. The guide covers six thematic categories: geospatial, earth observation/environment, meteorological, statistical, business/company ownership, and mobility data.

## Key Content

- **Legal Framework**: References EU Open Data Directive (2019/1024), EU Implementing Regulation 2023/138, Italian Legislative Decree 36/2006, and INSPIRE Directive (2007/2/CE)
- **General Requirements**: Machine-readable formats, API + bulk download, CC BY 4.0 or CC0 licensing, service quality criteria, contact point designation
- **Specific Requirements**: Detailed state-of-the-art assessments and action items for each of the 6 HVD categories and their constituent datasets
- **Metadata Standards**: DCAT-AP_IT for non-spatial data, RNDT for spatial data
- **Implementation Actions**: 10 numbered actions (#hvd1-#hvd10) covering format compliance, API implementation, service quality, documentation, licensing, and retroactive application

## Relevance

This source is critical for understanding the mandatory open data obligations of Italian public administrations. It connects to the wiki's existing coverage of data governance, metadata standards, and geospatial data infrastructure. The HVD regulation directly impacts how public sector data must be published, licensed, and made accessible, which has implications for data platform architecture and data quality management.

AgID (Agenzia per l'Italia Digitale) with contributions from a working group including ISTAT, IGM, Agenzia delle Entrate, ISPRA, and other public entities.
