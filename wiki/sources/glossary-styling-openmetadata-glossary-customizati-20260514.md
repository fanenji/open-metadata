# Glossary Styling | OpenMetadata Glossary Customization

**Source**: https://docs.open-metadata.org/v1.12.x/how-to-guides/data-governance/glossary/styling

**Official documentation** for the Glossary Styling feature in OpenMetadata v1.12.x. This page describes how to customize glossary terms with colors and icons for visual identification, enabling one-glance differentiation of related concepts and data assets.

## Key Points

- OpenMetadata supports glossary styling using colors and icons.
- Color-coding glossary terms enables one-glance visual identification of related concepts.
- When color-coded glossary terms are used for tagging data assets, it becomes easier to differentiate and identify data assets visually.
- The styling procedure is: Navigate to glossary term → Click ⋮ → Click "style" → Add icon image URL and/or change font color → Submit.
