# Best Practices for Glossary | Official Documentation

This official documentation page from OpenMetadata v1.12.x presents eight best practices for managing glossaries and classification tags. The guidance is prescriptive and opinion-based, focusing on scalability, governance, and usability of the controlled vocabulary system.

The eight practices cover: using hierarchical relationships (limited to three levels), attaching classification tags to glossary terms for auto-assignment, leveraging tier classification, using classification tags to simplify policies, optimizing display names for searchability, using glossary import/export for bulk edits, renaming over deleting glossary terms, and grouping similar concepts together.

The source emphasizes the dual role of Glossary and Classification as controlled vocabularies, and provides practical patterns for organizations to scale their data governance efforts by combining semantic meaning with data type definitions.
