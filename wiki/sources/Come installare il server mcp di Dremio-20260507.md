# Come installare il server mcp di Dremio

This source provides step-by-step instructions for installing and configuring the Dremio MCP server in a self-hosted deployment. It covers prerequisites (uv package manager, Dremio URI, Personal Access Token), cloning the official GitHub repository, creating configuration files for Dremio and Claude, and running the server in two operational modes: `FOR_DATA_PATTERNS` for natural language SQL queries and `FOR_SELF` for performance analysis. The source references official Dremio documentation and a LobeHub community guide.
