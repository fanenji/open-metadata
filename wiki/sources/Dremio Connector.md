# Dremio Connector

A note referencing a custom OpenMetadata connector for Dremio, hosted on GitHub by TIKI-Institut. The connector enables metadata ingestion from Dremio into OpenMetadata, which does not natively support Dremio as a data source.

## Key Links

- **Custom Connector Repository:** [TIKI-Institut/openmetadata-dremio-connector](https://github.com/TIKI-Institut/openmetadata-dremio-connector)
- **OpenMetadata Custom Connector Example:** [open-metadata/openmetadata-demo/custom-connector](https://github.com/open-metadata/openmetadata-demo/tree/main/custom-connector)
- **OpenMetadata Custom Connectors Documentation:** [docs.open-metadata.org](https://docs.open-metadata.org/latest/connectors/custom-connectors)
- **Video Tutorials:**
  - [Custom Connector Walkthrough (YouTube)](https://www.youtube.com/watch?v=ZvA4wuvINFA)
  - [Additional Custom Connector Video (YouTube)](https://www.youtube.com/watch?v=fDUj30Ub9VE)

## Relevance

This connector fills a gap in the OpenMetadata ecosystem by providing metadata ingestion support for [[dremio]], a data lakehouse query engine. It enables organizations using Dremio to manage metadata, schema, and lineage within [[openmetadata]].
