# Data Insights Report | OpenMetadata Reporting Guide

This source is the official OpenMetadata v1.12.x documentation page for the Data Insights Report. It provides a detailed specification of the four report categories available to administrators: Data Assets, App Analytics, Key Performance Indicators (KPIs), and Cost Analysis (SaaS-only). The document defines each metric within these reports, including Total Data Assets, Percentage of Data Assets with Description/Owners, Service-level coverage breakdowns, Tier distribution, Most Viewed Data Assets, Daily Active Users, and Used vs. Unused Assets analysis. It also documents the cross-cutting filtering options (Teams, Data Tiers, Time Filter) and explicitly marks the Cost Analysis section as exclusive to OpenMetadata SaaS users.

This source complements the existing [[data-insights-application-troubleshooting]] page by describing what the reports should display when functioning correctly, and extends the [[openmetadata-insights]] concept with granular report definitions.
