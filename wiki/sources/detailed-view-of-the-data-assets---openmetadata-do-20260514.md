# Detailed View of the Data Assets - OpenMetadata Documentation

**Source:** [https://docs.open-metadata.org/v1.12.x/how-to-guides/data-discovery/details](https://docs.open-metadata.org/v1.12.x/how-to-guides/data-discovery/details)

This official OpenMetadata v1.12.x documentation page describes the detailed view of data assets, covering the top panel metadata display, version history and other controls, and the tab-based organization of the data asset details page. It enumerates 13+ tabs (Schema, Activity Feeds & Tasks, Sample Data, Queries, Data Observability, Lineage, Custom Properties, Config, Details, Executions, Features, Children) and specifies which asset types (Table, Topic, Dashboard, Pipeline, ML Model, Container) each tab applies to.

Key content includes:
- Top panel: Source, Owner, Tier, Type, Usage, Description.
- Top-right controls: Tasks, Version History, Follow, Share, Announcements, Rename, Delete.
- Tab availability matrix per asset type.
- Schema Tab: columns, types, descriptions, tags, glossary terms, frequently joined tables (Tables, Topics, Containers).
- Activity Feeds & Tasks Tab: tasks and mentions (all asset types).
- Sample Data Tab: ingested sample data (Tables, Topics).
- Queries Tab: SQL queries with duration and usage info (Tables only).
- Data Observability Tab: Table Profile, Column Profile, Data Quality sub-tabs (Tables only).
- Lineage Tab: end-to-end lineage, configurable layers, manual no-code editing (all asset types).
- Custom Properties Tab: view/edit custom property values (all asset types).
- Config Tab: topic configuration (Topics only).
- Details Tab: chart info for Dashboards, Hyper Parameters/Model Store for ML Models.
- Executions Tab: pipeline status as chronological list or tree (Pipelines only).
- Features Tab: ML Model features with type, algorithm, sources, glossary terms, tags (ML Models only).
- Children Tab: container contents (Containers only).
