# Overview of Announcements | Official Documentation - OpenMetadata Documentation

This source is the official OpenMetadata v1.12.x documentation page for the Announcements feature. It describes how announcements are scheduled, time-bound notifications about upcoming changes to data assets, displayed as banners on the data asset details page and in the Activity Feed. The source provides practical guidance on scheduling announcements well in advance and ensuring all backward incompatible changes are announced. It also covers announcement details (creator, data asset, scheduled date range), user interactions (emoji reactions and replies), and the integration with Alerts for external notifications via Email, Chat, Slack, MS Teams, and Webhooks.
