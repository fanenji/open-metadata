# Freelens Kubernetes Client Setup

This source is a Gemini conversation response explaining how to configure the Freelens Kubernetes IDE on macOS. It covers two installation methods (direct download and Homebrew), automatic cluster discovery via `~/.kube/config`, manual cluster addition, and a detailed explanation of the kubeconfig file structure (clusters, users, contexts). The source serves as a practical, how-to guide for setting up a Kubernetes client on macOS.
