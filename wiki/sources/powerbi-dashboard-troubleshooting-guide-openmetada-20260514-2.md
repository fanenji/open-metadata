# PowerBI Dashboard Troubleshooting Guide | OpenMetadata Support

This source is the official OpenMetadata v1.12.x troubleshooting guide for the PowerBI dashboard connector. It documents three common failure scenarios: Workflow Deployment Error (partial failure where the ingestion pipeline entity is created but no workflow runs), Debug Logging procedure for enabling verbose logs, and generic Permission Issues that defer to connector-specific prerequisites. The guide is procedural and does not provide root-cause analysis or PowerBI-specific permission requirements.
