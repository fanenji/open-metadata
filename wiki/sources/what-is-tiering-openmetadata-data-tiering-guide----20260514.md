# What is Tiering | OpenMetadata Data Tiering Guide

Official OpenMetadata documentation page explaining the tiering concept for data classification. Tiering is a system for ranking data assets by business importance using a five-tier scale (Tier 1 = most important, Tier 5 = least important). The guide covers the tier impact/usage matrix, the "start with extremes" methodology, and how Data Insights helps identify Tier 5 unused datasets for periodic decluttering.

**Source:** [OpenMetadata Documentation v1.12.x](https://docs.open-metadata.org/v1.12.x/how-to-guides/data-insights/tiering)
