# Configure the Data Insights Report - OpenMetadata Documentation

This official documentation page provides a brief, step-by-step guide for configuring the Data Insights email report in OpenMetadata v1.12.x. It covers the installation of the Data Insights Report application via the Application Framework (Settings >> Applications >> Add Apps), authorization, recipient selection (Admins and/or Teams), and scheduling the email distribution at a desired frequency. The page also mentions the ability to view the report history after configuration.

The source is minimal and assumes that [[data-insights-ingestion]] has already been set up as a prerequisite. It does not cover email infrastructure requirements, error states, or detailed troubleshooting for the email report.

## Key Points

- The Data Insights Report is an installable application within the [[application-framework]].
- Prerequisite: Data Insights Ingestion must be running.
- Recipients can be Admins and/or Teams.
- The report can be scheduled at a desired frequency.
- After configuration, the schedule and recipients can be modified, and the report history is viewable.
