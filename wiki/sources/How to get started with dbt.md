# How to get started with dbt

A comprehensive resource hub for understanding dbt basics and getting started with the framework. The article covers dbt Core (the open-source framework), its foundational ELT paradigm, key concepts (models, sources, Jinja templating, macros, ref/source macros), dbt entities (tests, seeds, incremental models, snapshots, docs, exposures, metrics, analyses), and the analytics engineering role. It emphasizes that dbt is a framework for organizing SQL transformations, not a scheduler, and that it only performs the T in ELT. The article also provides curated external resources including conference takeaways, online courses, and community projects.
