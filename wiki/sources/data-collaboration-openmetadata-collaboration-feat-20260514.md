# Data Collaboration | OpenMetadata Collaboration Features

Official OpenMetadata documentation page (v1.12.x) providing an overview of the three pillars of data collaboration: Conversation Threads, Tasks, and Announcements. The page describes how these features enable teams to break information silos, share organizational knowledge, and collaborate directly within the platform.

## Key Points

- **Core Claim:** OpenMetadata's collaboration features are a catalyst for breaking information silos and enhancing data understanding.
- **Three Pillars:** Conversation Threads (discussions around data assets and tags), Tasks (structured requests for descriptions, tags, and glossary term approval), and Announcements (scheduled team-wide notifications about upcoming changes).
- **Audience:** Data teams and data users.
- **Relationship to Activity Feeds:** All three collaboration features surface in the Activity Feed, which serves as the central UI component.
