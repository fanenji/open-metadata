# Guide to Searching Data Using Hierarchical View - OpenMetadata Documentation

**Source**: [OpenMetadata v1.12.x Documentation](https://docs.open-metadata.org/v1.12.x/how-to-guides/data-discovery/hierarchy)

A brief procedural guide describing the Hierarchy View feature in OpenMetadata, which organizes data assets in a structured, layered manner based on relationships and dependencies. The guide covers two steps: accessing the Hierarchy View from the OpenMetadata interface, and navigating the hierarchical structure by expanding/collapsing levels to drill down into specific assets or view parent assets. Clicking an asset in the hierarchy opens its detailed information, metadata, and related data connections.

This is a thin documentation page with no technical depth, examples, or screenshots. It serves as a basic introduction to hierarchical data discovery as an alternative to search-based discovery.
