# The ultimate guide to dbt

A comprehensive visual guide to dbt, created by [[Taylor Brownlow]] at [[Count.co]], intended as a one-stop-shop for learning about dbt from beginner to expert level. The guide is presented as an interactive canvas rather than a traditional blog post, covering dbt's purpose, key concepts, and reference materials.

The author spent a week researching dbt's documentation, over 40 articles, community threads, and expert consultations to synthesize the material. The guide is designed as a [[living-guide]] — a continuously updated resource that accepts community contributions.

The source document itself contains only images and metadata pointing to the external canvas at `https://count.co/canvas/JpkaYdqr9oN`. The substantive content of the guide is hosted externally and was not captured in this markdown file.

## Key Topics Covered (per the canvas map)

- Intro and context materials
- Key concepts of dbt
- Reference guides and resources
