# Overview of Announcements | Official Documentation

This source is the official OpenMetadata v1.12.x documentation page for the Announcements feature. It describes the purpose, creation workflow, scheduling, display mechanisms, and interaction capabilities of announcements. The page positions announcements as a solution to the organizational challenge of communicating upcoming data changes (deprecation, deletion, schema changes) to the entire team in a timely manner.

Key points covered:
- Announcements are scheduled, time-bound notifications displayed as banners on data asset detail pages and in the Activity Feed.
- They have a start and end date defining the display window.
- Users can react with emojis and reply via threaded comments.
- Alerts can be configured to extend notification reach to Email, Chat, Slack, MS Teams, and Webhooks.
- The documentation recommends scheduling announcements well in advance, especially for backward-incompatible changes like column deletion.
