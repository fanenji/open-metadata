# Announcing OpenMetadata 1.12

This source announces the release of OpenMetadata 1.12, highlighting key advancements in AI-driven metadata management, simplified cloud-native deployment, and support for open standards.

Key features discussed:
- **AI SDK**: Programmatic access to the metadata knowledge graph.
- **Data Quality Test Library**: Reusable, SQL-based, parameterized test templates.
- **Kubernetes Orchestrator**: Removal of the Apache Airflow dependency for a streamlined 3-component architecture.
- **Open Standards**: Support for ODCS 3.1 (Data Contracts) and OpenLineage.
- **Semantic Search**: Leveraging OpenSearch for vector embeddings.
