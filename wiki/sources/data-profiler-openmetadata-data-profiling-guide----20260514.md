# Data Profiler | OpenMetadata Data Profiling Guide

**Source:** [OpenMetadata Documentation v1.12.x](https://docs.open-metadata.org/v1.12.x/how-to-guides/data-quality-observability/profiler)

## Overview

This is a high-level navigation/index page for the Data Profiler section of the OpenMetadata documentation (v1.12.x). It introduces the profiler as a tool for understanding data shape and validating assumptions by capturing table usage statistics over time. The page serves as a gateway to deeper sub-pages covering the Profiler Workflow, Metrics, Auto PII Tagging, and Spark Engine integration.

## Key Points

- The profiler helps understand the shape of data and quickly validate assumptions.
- It captures table usage statistics over a period of time as part of profiler ingestion.
- Data profiles enable checking for null values in non-null columns, duplicates in unique columns, and understanding column data distributions through descriptive statistics.
- The page links to sub-pages for: Profiler Workflow, Metrics, Auto PII Tagging, and Spark Engine.
