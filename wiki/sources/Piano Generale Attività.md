# Piano Generale Attività

This document defines the master integration and development plan for the Data Platform (DP). It outlines four main work streams:

1. **Integration Middleware Development (5.1):** Build interoperability middleware connecting platform components: orchestration, data analysis, virtualization, exploration, the regional Data Catalog, and authentication. Two pilot use cases (RQA and geo data ingestion) are identified for validation.

2. **Workload Migration (5.2):** Migrate existing workloads to the new Data Platform.

3. **Catalog Integration & New Subsystems (5.3/5.4):** Integrate the platform's Data Catalog with an external Open Data Catalog (Catalogo OD). Build new subsystems: Dynamic Download (Download Dinamico), OpenData API, reporting system, and cartographic components for data exploration. Migrate the Open Data catalog onto the Data Platform.

The document is a high-level planning outline with no timeline, resource allocation, or success criteria. It serves as the architectural roadmap for the wiki's existing tool-specific content.
