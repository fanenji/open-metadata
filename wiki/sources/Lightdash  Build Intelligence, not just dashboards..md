# Lightdash | Build Intelligence, not just dashboards.

**Source:** [https://www.lightdash.com/](https://www.lightdash.com/)

This is the marketing landing page for **Lightdash**, an open-source, AI-native BI platform. The page positions Lightdash as a solution to the bottleneck of manual analytics, enabling data teams to ship analytics at the speed of code.

## Key Claims

- **Agentic BI:** Lightdash is described as the only open-source AI-native BI platform that lets AI build, refactor, and ship analytics in minutes.
- **Dashboards-as-Code:** The platform enables version-controlled, code-driven dashboard management, aligning with software engineering workflows.
- **dbt Integration:** Lightdash connects directly to dbt, allowing metrics to be defined once and reused across the organization.
- **Looker Alternative:** The page explicitly targets Looker's pricing model, offering unlimited user seats and a more developer-friendly approach.

## Features Highlighted

- Unlimited user seats (cost savings vs. Looker)
- Data governance via natural language queries
- Usage analytics for tracking BI adoption
- Developer-friendly workflows (CLI, code-based analytics)
- Enterprise-grade security (SOC2 Type II, HIPAA compliant)
- Lightning-fast support from data practitioners

## Evidence

The page relies on anecdotal customer testimonials and a "Happiness Curve" diagram suggesting a 5-day path to 10x productivity. No quantitative benchmarks or independent studies are provided. The strength of evidence is weak, as the source is a marketing page.
