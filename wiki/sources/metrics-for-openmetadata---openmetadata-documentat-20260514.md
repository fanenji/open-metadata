# Metrics for OpenMetadata - OpenMetadata Documentation

This source is the official OpenMetadata documentation page for the Metrics entity (v1.12.x). It defines metrics as a first-class governance entity under the Govern section, covering key properties (name, display name, description, expression/formula, granularity, metric type, unit of measurement, SQL query, owner), the UI creation workflow (5 steps), metric lineage and dependencies (Database Table → Metric → Pipeline → Dashboard), management capabilities (versioning, linking, monitoring), and best practices. The source includes an example JSON schema and positions metrics as distinct from Data Quality tests, under the Data Governance category.
