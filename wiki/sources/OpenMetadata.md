# OpenMetadata Map of Content

This is a Map of Content (MOC) for the OpenMetadata platform, serving as a central reference point for the data platform's metadata management strategy.

## Official Documentation

- [OpenMetadata Documentation](https://docs.open-metadata.org/latest) — The official documentation for the latest version of OpenMetadata.

## Cloud vs. Open-Source Comparison

- [OpenMetadata Cloud vs. Open-Source Comparison](https://www.getcollate.io/comparison) — A detailed comparison of the managed cloud offering and the self-hosted open-source version, maintained by Collate.

## Significance

This MOC confirms OpenMetadata as the recommended open-source metadata platform for the data platform. The key architectural decision point is the choice between the managed Cloud version and the self-hosted Open-Source version, which affects deployment, maintenance, and feature availability.
