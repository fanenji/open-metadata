# Conversation Threads | OpenMetadata Collaboration Guide

This source is the official documentation page for Conversation Threads in OpenMetadata v1.12.x. It describes how threaded discussions enable users to ask questions and discuss data assets directly within the platform. The feature was introduced in v0.9.0 with the ability to reply and create threads around activities on any data asset. In v0.11.0, it was extended to support Tasks and emoji reactions.

Users can start conversations around descriptions, column descriptions, tags, announcements, and glossary terms by clicking a chat icon. Replies and emoji reactions are supported by hovering over a conversation post.

This source is a brief feature overview and does not contain deep technical details or architectural information.
