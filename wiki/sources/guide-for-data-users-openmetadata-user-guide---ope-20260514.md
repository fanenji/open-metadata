# Guide for Data Users | OpenMetadata User Guide

This source is the official OpenMetadata v1.12.x user guide for data users. It provides a high-level index of user-facing features and workflows, covering the landing page, data asset management, collaboration, and governance from the perspective of a non-administrative data consumer.

## Summary

The guide is structured as a navigation index linking to detailed procedures for:

- **Understanding the Basics of OpenMetadata** — Landing page and widgets (Activity Feeds, Data Insights, Announcements).
- **Overview of Data Assets** — Core objects (tables, topics, dashboards, pipelines) and their details.
- **Adding Descriptions using Markdown** — Documenting data assets.
- **Requesting for Descriptions** — Collaborative description requests via Tasks.
- **Assigning or Changing Data Ownership** — Managing data asset ownership.
- **Following a Data Asset** — Subscribing to updates on specific assets.
- **Adding Tags** — Manual tagging and auto-PII classification.
- **Requesting for Tags** — Collaborative tag requests via Tasks.
- **Creating a Personal Access Token** — Authentication for API/CLI access.
- **Adding Glossary Terms** — Business vocabulary classification.
- **Creating Custom Properties** — Extending data models with user-defined fields.
- **Customizing OpenMetadata** — Persona and landing page customization.
- **Overview of Announcements** — Communication mechanism for data changes.
- **Creating an Announcement** — Step-by-step announcement workflow.
- **Data Asset Versioning** — Major/minor change tracking with version history.
- **Deleting a Data Asset** — Soft and hard deletion modes.
- **OpenMetadata Browser Extension** — Peripheral tool for quick access.

## Key Insights

- The guide emphasizes the "Data User" perspective, distinct from Admin or Steward roles, filling a gap for end-user workflows.
- It consolidates features documented separately across the wiki (tags, ownership, descriptions, announcements) into a single user-facing narrative.
- The "My Data" page is accessible from anywhere by clicking the OpenMetadata logo, providing a quick navigation shortcut.
- Auto-PII classification is listed under "How to Add Tags," which may cause confusion about whether tagging is manual or automated.
