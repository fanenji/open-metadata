# Handling Sensitive Data in Your Data Platform

A practical guide by Mariusz Kujawski covering techniques for handling sensitive data in a modern data platform, including identification, governance, and GDPR compliance. The article emphasizes that the first question should always be whether sensitive data is actually needed, and provides a hierarchy of techniques from removal to clear text storage with strict access controls. It covers masking, tokenization, encryption, and hashing with Databricks SQL examples, and discusses automated data classification and attribute-based access control (ABAC) in Unity Catalog.
