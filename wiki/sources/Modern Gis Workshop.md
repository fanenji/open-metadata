# Modern GIS Workshop

A hands-on workshop presented at the I-Guide Forum 2023 at Columbia University, covering five modern geospatial technologies: [[GeoParquet]], [[PostGIS]], [[DuckDB]], [[dbt]], and [[h3-geospatial-indexing|H3]]. The workshop demonstrates how to build a modern GIS workflow using open-source tools, replacing traditional GIS software with a cloud-native, analytical stack.

## Structure

The workshop is organized into five parts, each with accompanying code and guides:

1. **Part 1: GeoParquet** — Working with the columnar geospatial format for cloud-native workflows.
2. **Part 2: PostGIS** — Spatial queries and analysis using the PostgreSQL spatial extension.
3. **Part 3: DuckDB** — Lightweight analytical queries with DuckDB's geospatial capabilities.
4. **Part 4: dbt** — Data transformation and testing for geospatial data pipelines.
5. **Part 5: H3** — Hexagonal grid indexing for scalable spatial analytics.

## Significance

This workshop validates the modern geospatial stack documented in this wiki, demonstrating practical integration of [[GeoParquet]], [[DuckDB]], [[dbt]], and [[H3]] alongside [[PostGIS]] as a hybrid approach combining traditional spatial databases with cloud-native tools.
