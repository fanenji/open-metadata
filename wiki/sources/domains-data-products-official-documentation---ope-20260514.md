# Domains & Data Products | Official Documentation - OpenMetadata Documentation

This source is the official OpenMetadata v1.12.x documentation page for Domains and Data Products. It introduces the Data Mesh paradigm within OpenMetadata, explaining how Domains enable decentralized data ownership and management, and how Data Products package related data assets for discoverability and usability. The page covers key principles, benefits, setup workflows, and the Domain-Only View feature.

## Key Topics

- **Domains**: Organizational units aligned with business capabilities, managed by cross-functional teams.
- **Data Products**: Packaged data assets (datasets, models, pipelines) produced by domains.
- **Data Mesh**: Architectural foundation for decentralized data management.
- **Domain-Only View**: UI feature for focused navigation within a single domain.
- **Implementation Workflow**: Steps for setting up domains and creating data products.

## Relevance

This source is foundational for understanding how OpenMetadata supports a Data Mesh architecture. It introduces two new core concepts (Domains and Data Products) that extend the platform's data governance capabilities beyond the existing team hierarchy and glossary structures.
