# How to install the Dremio MCP server

This source provides a procedural guide for installing and configuring the Dremio Model Context Protocol (MCP) server in a self-hosted environment. It covers prerequisites like the `uv` package manager, the use of Personal Access Tokens (PAT), and specific configuration modes for AI clients like Claude.
