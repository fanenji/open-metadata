# Context is Important, Metadata Provides It

An article by Nick Acosta discussing the necessity of metadata platforms in providing the organizational context required for AI agents and LLMs to function reliably, even when using the Model Context Protocol (MCP).

The author argues that while MCP solves the integration/connectivity problem, it does not solve the context/meaning problem. A metadata platform (like [[openmetadata]]) provides the "Unified Knowledge Graph" that makes AI "organizationally-aware."
