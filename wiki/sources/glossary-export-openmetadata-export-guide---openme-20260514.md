# Glossary Export | OpenMetadata Export Guide

**Source:** https://docs.open-metadata.org/v1.12.x/how-to-guides/data-governance/glossary/export

This official OpenMetadata v1.12.x documentation page describes the **Glossary Export** feature, a UI-driven tool for downloading a glossary's terms as a CSV file. The process is simple: navigate to Govern > Glossary, click the ⋮ icon on the desired glossary, and select Export. If the glossary contains terms, they are exported as a populated CSV file. If the glossary is empty, a blank CSV template is downloaded instead.

The page also references two related features: **Bulk Import** (the complementary operation for uploading glossary terms via CSV) and **Glossary Approval Workflow** (a review and approval process for glossary terms). These are linked from the bottom of the page for further exploration.

This source fills a gap in the wiki by documenting the export counterpart to the existing [[how-to-add-glossary-terms]] import procedure. It is directly related to the [[glossary-terms]] entity and the broader [[classification-tags]] governance concept.
