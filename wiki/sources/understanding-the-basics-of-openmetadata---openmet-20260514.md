# Understanding the Basics of OpenMetadata

**Source:** [OpenMetadata Documentation v1.12.x](https://docs.open-metadata.org/v1.12.x/how-to-guides/guide-for-data-users/basics-openmetadata)

This page is the official introductory guide for data users in OpenMetadata v1.12.x. It describes the 'My Data' landing page and its constituent widgets: the Activity Feed Widget, My Data Widget, Key Performance Indicators (KPI) Widget, Total Data Assets Widget, and the right-side panel containing Announcements, Following, and Recent Views. It serves as the primary entry point for users to understand the platform's user interface and collaboration features.

## Key Content

- **My Data Page** — The default landing page after login, composed of multiple widgets providing a single-pane view of owned/followed assets, collaboration updates, and data insights.
- **Activity Feed Widget** — Displays all activities related to owned, followed, or mentioned data assets, plus @mentions and open tasks.
- **My Data Widget** — Lists all data assets owned by the current user; directs users to the Explore page to claim unowned assets.
- **KPI Widget** — Shows ownership, description, and tiering coverage metrics; accessible to Admins by default, with potential extension to other users.
- **Total Data Assets Widget** — 14-day trend visualization of total data assets by type (Tables, Dashboards, Databases, etc.).
- **Right-Side Panel** — Contains Announcements, Following, and Recent Views for secondary navigation and awareness.
