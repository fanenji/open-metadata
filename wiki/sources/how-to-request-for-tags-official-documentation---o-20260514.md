# How to Request for Tags | Official Documentation - OpenMetadata Documentation

This source is the official OpenMetadata v1.12.x documentation page for the "How to Request for Tags" guide. It covers two main features: the tag request workflow via Tasks and conversations around tags.

## Key Content

- **Tag Request Workflow**: Users can request tag updates by clicking the `?` icon next to tags. This creates a Task with pre-populated details (Title auto-populated, Assignees, Update Tags with three tabs: Current, New, Difference). The task appears in the Activity Feeds & Tasks tab for the data asset. Assignees can Accept, Edit & Accept, or Comment on the suggestion.
- **Conversations around Tags**: Users can start a threaded discussion by clicking the Conversation icon next to a tag. Supports @mentions (users/teams), #mentions (data assets), replies, reactions, editing, and deletion.

## Relevance

This source provides the official procedural details for the [[tag-request-workflow]] and [[conversations-around-classification]] concepts, confirming the Task-based nature of tag requests and the three-tab diff interface. It also documents the `?` icon entry point for initiating tag requests.
