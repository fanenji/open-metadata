# ArcKit — AI Toolkit for Solution & Enterprise Architects

An article by David R. Oliver describing **ArcKit**, an open-source, AI-assisted toolkit designed to automate the generation of architectural governance artifacts (ADRs, requirements, compliance assessments) using AI coding assistants like Claude Code.

The article highlights the "friction problem" in architecture governance and proposes a solution through **knowledge compounding**, where structured, AI-readable Markdown files create increasing value over time. It also details a security model based on **skills** (constrained generation), **hooks** (automated safety nets), and **plugins**.
