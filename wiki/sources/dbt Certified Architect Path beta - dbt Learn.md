# dbt Certified Architect Path [beta] - dbt Learn

This is the official beta learning path published by dbt Labs on their learning platform (learn.getdbt.com) to prepare for the **dbt Certified Architect Examination**. The path is structured into six milestones covering the full scope of dbt Cloud architecture, from connection setup to exam registration.

## Structure

The learning path consists of six milestones:

1. **Setting up dbt Connections** — Configuring data platforms (Snowflake, Databricks, BigQuery, Redshift) and Git integration.
2. **Configuring and Managing Projects** — dbt Cloud environments, Git branching strategies (Direct/Indirect Promotion), and advanced deployment (CI/CD, job orchestration).
3. **Security and Monitoring** — SSO, RBAC, licenses, permissions, and webhooks for automation.
4. **Governance and Discovery** — dbt Mesh for cross-project dependencies and dbt Catalog for documentation and discovery.
5. **Exam Preparation** — Advanced setup workshop, study guide, and pro tips.
6. **Exam Registration** — Final step to register for the exam.

## Key Takeaways

- The certification covers five technical domains: Connections, Project Configuration & Environments, Security & Monitoring, Governance & Discovery, and Exam Preparation.
- The path is marked as "beta" — content is actively being updated based on learner feedback.
- dbt Mesh and dbt Catalog are positioned as the governance and discovery layer for enterprise dbt Cloud deployments.
- SSO/RBAC and webhooks are core security and automation topics for enterprise administration.
