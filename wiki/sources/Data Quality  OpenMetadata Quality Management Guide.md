# Data Quality | OpenMetadata Quality Management Guide

This source is the official OpenMetadata documentation page on data quality management. It describes the platform's built-in capabilities for monitoring data completeness, freshness, and accuracy through native tests, alerting, health dashboards, and resolution workflows. The guide positions OpenMetadata as a no-code, extensible data quality platform integrated directly into the metadata catalog.

Key features documented:
- **Native tests** — built-in assertions for all supported database connectors at table and column levels.
- **Alerting system** — notifications triggered on test failure.
- **Health dashboard** — real-time tracking of test failures for prioritization.
- **Resolution workflow** — process to inform data consumers when tests are resolved.
- **Extensibility** — ability to adapt data quality to custom needs.

The source includes references to video tutorials demonstrating the profiler and data quality features.
