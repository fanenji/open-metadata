# Overview of Classification | Official Documentation

This source is the official OpenMetadata documentation page (v1.12.x) providing a high-level overview of the Classification system. It defines Classification as a tag-based system for categorizing data assets by type (e.g., sensitive/non-sensitive, PII/non-PII), distinct from defining semantics or meaning. The page introduces the critical distinction between Classification tags (mutually exclusive) and Categorization tags (not mutually exclusive), the Mutually Exclusive Tags configuration option, and the role of Classification in policy enforcement, search, and browsing. It also covers System Classifications (PersonalData, PII, Tiers) and mentions the Classification APIs for automation.
