# How to Classify Data Assets | Official Documentation

Official procedural guide for applying classification tags to data assets in OpenMetadata v1.12.x. Covers the 3-step UI workflow for tagging assets from the Explore page, the reverse-lookup capability from Govern > Classification to see tag usage counts and drill down to tagged assets, and introduces the broader classification framework including Tiers and System Classification. Also references the collaborative tag request workflow where users can request and discuss tag changes within the platform.

## Key Points

- **3-step tagging workflow**: Navigate to asset on Explore page → search/select tags → save with checkmark.
- **Reverse lookup**: Govern > Classification shows all tags with usage counts; click the count to view tagged assets.
- **Classification subsystems**: General classification tags, Tiers (for data importance ranking), and System Classification (built-in tags).
- **Collaborative governance**: Users can request tag updates and discuss them within OpenMetadata.
