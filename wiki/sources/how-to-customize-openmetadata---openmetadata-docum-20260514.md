# How to Customize OpenMetadata - OpenMetadata Documentation

**Source:** [OpenMetadata Documentation v1.12.x](https://docs.open-metadata.org/v1.12.x/how-to-guides/guide-for-data-users/theme)

**Clipped:** 2026-05-14

## Summary

This official documentation page describes how to personalize the OpenMetadata platform's appearance with company branding and custom colors. Users can navigate to **Settings > Preferences > Theme** to upload a company logo, monogram, and favicon, and to set a primary color and info color. Changing the primary color dynamically adjusts the appearance of buttons, navigation panes, data assets, tab headers, and more. The info color is reflected on selected data assets. A one-click revert to the original theme is available.

## Key Points

- **Navigation:** Settings > Preferences > Theme
- **Customizable elements:** Company logo, monogram, favicon, primary color, info color
- **Dynamic UI adjustment:** Primary color change automatically updates multiple UI components
- **Revert:** One-click reset to original theme
- **Scope:** Global customization affecting all users, distinct from persona-based personalization
