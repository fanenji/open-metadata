# Getting Started with OpenMetadata for Data Cataloging

This source is the official OpenMetadata "Getting Started" guide (v1.12.x). It provides a high-level overview of the platform's architecture, key features, and setup process. The document positions OpenMetadata as a unified platform for data discovery, observability, and governance, emphasizing a 30-minute setup time and a library of 90+ turnkey connectors.

## Key Claims

- OpenMetadata centralizes all data context to help build high-quality data and AI assets.
- Metadata from various sources is organized into a [[unified-metadata-graph]], providing a single source of truth.
- The platform supports both technical and non-technical users through native collaboration features.
- Key feature categories include Discovery, Lineage, Observability, Quality, Collaboration, Governance, and Insights.

## Limitations

- The document is a marketing/onboarding guide with no empirical evidence or benchmarks.
- The "30-minute setup" claim is aspirational and likely underestimates real-world complexity.
- Feature descriptions are high-level and lack implementation details.

## Relevance

This source serves as a foundational overview for the wiki, introducing core concepts and features that are expanded upon in dedicated entity and concept pages.
