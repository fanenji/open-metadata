# pygeoapi - an OGC API to geospatial data

pygeoapi is a Python server implementation of the OGC API suite of standards. The project emerged as part of the next generation OGC API efforts in 2018 and provides the capability for organizations to deploy a RESTful OGC API endpoint using OpenAPI, GeoJSON, and HTML. pygeoapi is open source and released under an MIT license.

The project is certified OGC Compliant and is an OGC Reference Implementation for multiple OGC API standards including Features, Tiles, EDR, and Processes. It is an OSGeo Project and has been featured at FOSS4G conferences.

Key features include:
- OGC Compliant certification and OGC Reference Implementation status
- Easy installation via pip or git
- Easy deployment via UbuntuGIS or official Docker image
- Flexible plugin framework for custom data sources, output formats, and processes
- Deployment options via Flask, Django, or any Python framework

The document provides a 5-minute installation guide and overview of the project's capabilities as a standards-compliant geospatial API server.
