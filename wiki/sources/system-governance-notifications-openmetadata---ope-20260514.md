# System & Governance Notifications | OpenMetadata

Official documentation for the System & Governance Notifications feature in OpenMetadata v1.12.x. This source describes a dedicated alert mechanism for metadata changes, governance events, and collaboration activities, distinct from Data Observability Alerts which focus on data quality and pipeline failures.

The source covers the 5-step setup workflow: navigating to the Notifications page, creating a new alert, selecting the metadata entity to monitor, configuring optional filters (Owner, Entity FQN, Event Type, Updater Name, Domain, Updater Is Bot), and selecting internal (Admins, Followers, Owners, Teams, Users) or external (Email, Slack, MS Teams, Google Chat, Generic Webhooks) destinations. It also documents the Include/Exclude toggle logic for filter conditions.
