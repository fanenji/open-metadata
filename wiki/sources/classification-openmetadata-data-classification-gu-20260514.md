# Classification | OpenMetadata Data Classification Guide

**Source:** [OpenMetadata Documentation v1.12.x](https://docs.open-metadata.org/v1.12.x/how-to-guides/data-governance/classification)

This page is the top-level navigation hub for the Classification feature in OpenMetadata's Data Governance section. It introduces Classification as a controlled vocabulary for labeling data, distinct from but complementary to Glossary. The page links to detailed sub-pages covering: how to classify data assets, how to request classification tags, sample data handling with PII tags, auto-classification workflow, tiers, and best practices.

## Key Points

- **Classification** is a controlled vocabulary — an organized arrangement of words and phrases to define terminology for organizing and retrieving information.
- **Purpose:** Helps define the type of data (e.g., PII, sensitive, public).
- **Distinction:** Classification is for technical/governance labeling; Glossary is for business metadata.
- **Sub-features:** System tags, mutually exclusive tags, PII tags, tiers, auto-classification.
- **Related:** Glossary, Domains & Data Products, Data Governance overview.
