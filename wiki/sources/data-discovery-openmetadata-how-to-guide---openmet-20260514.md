# Data Discovery | OpenMetadata How-To Guide

**Source:** [OpenMetadata Documentation v1.12.x](https://docs.open-metadata.org/v1.12.x/how-to-guides/data-discovery)

A high-level overview page from the official OpenMetadata documentation describing the data discovery capabilities of the platform. The page frames data discovery as a core challenge — finding relevant data among thousands of datasets — and presents OpenMetadata's solution: a single catalog aggregating rich metadata with faceted search.

The guide outlines three primary discovery strategies: keyword search, data associations (frequently joined tables and lineage), and complex queries via advanced search. It lists the supported asset types (tables, topics, dashboards, pipelines, ML models, containers, glossaries, tags) and mentions support for detailed metadata on components (columns, charts) including complex data types. The page also references metadata versioning and lineage as tools for understanding data evolution.

This is a non-technical, introductory page with no procedural steps, configuration details, or empirical data. It serves as a feature overview rather than a deep technical reference.
