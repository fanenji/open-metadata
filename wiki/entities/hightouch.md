# Hightouch

A [[reverse-etl-pattern|Reverse ETL]] tool that syncs data from cloud data warehouses (e.g., [[snowflake]], [[bigquery]], [[redshift]]) back to operational SaaS tools like [[salesforce]], [[zendesk]], and [[hubspot]]. Hightouch is one of the leading tools in the reverse ETL space, alongside [[census]], enabling business users to access enriched data in their primary tools.
