# Apache Superset

Apache Superset is an open-source data exploration and visualization platform, serving as the business intelligence (BI) layer for the data platform.

## Infrastructure Sizing

The recommended deployment is a single node with 2 vCPU, 8 GB RAM, and 50 GB storage.
