# Backslash Security

Backslash Security is a security research organization that identified "overscoping" as the most pervasive default risk across published [[model-context-protocol]] (MCP) servers. Their research found that excessive permissions enable OS command injection and path traversal attacks.

## Key Findings
- Identified overscoping as the most pervasive default risk in MCP servers
- Documented how excessive permissions enable command injection and path traversal
- Their findings are a primary source for the [[mcp-attack-surface]] concept
