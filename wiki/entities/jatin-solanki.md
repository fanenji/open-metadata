# Jatin Solanki

Jatin Solanki is the founder of [[decube]], a data trust platform for the GenAI era. He is the author of the "Data Contracts: Implementation Guide," which presents a process-oriented approach to implementing [[data-contract]]s at scale. His work emphasizes that data contracts are primarily a cultural and process challenge rather than a technical one.

Solanki's guide provides an 8-step implementation process, a real-life scenario illustrating the need for data contracts, and a detailed CI/CD enforcement pattern using Git and YAML. He credits [[Andrew Jones]] as the creator of the data contracts concept.
