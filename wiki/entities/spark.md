# Apache Spark

Apache Spark is a unified, open-source distributed processing engine for large-scale data analytics. It provides batch and stream processing capabilities and is used for complex transformations, large-scale analysis, and machine learning tasks within the Data Platform.

## Infrastructure Sizing

The recommended deployment consists of 4 nodes, each with 4 vCPU, 16 GB RAM, and 100 GB storage.
