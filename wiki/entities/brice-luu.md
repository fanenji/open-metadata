# Brice Luu

Author of the article "Modern Data Modeling: Start with the End?" published on Advent of Data in December 2022. Brice Luu presents a practical, bottom-up methodology for structuring dbt projects, advocating for an iterative approach that prioritizes delivering value quickly over upfront design.

## Key Contributions

- Formulated the **Base → Core → Usecases** three-layer dbt project structure
- Popularized the **rule of three** heuristic for progressive refactoring in data modeling
- Articulated the **agile analytics engineering** philosophy
- Documented the `generate_schema_name` macro pattern for environment-aware schema mapping
