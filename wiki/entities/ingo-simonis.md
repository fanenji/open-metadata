# Ingo Simonis

Author of the comprehensive comparison "FlatGeoBuf vs. GeoParquet vs. Apache Iceberg vs. OGC API-Features: A Cloud Geospatial Comparison" published on LinkedIn in July 2025. The article provides a detailed technical analysis of four modern approaches to storing and serving geospatial vector data, with practical recommendations for format selection.

## Contributions

- Systematic comparison of FlatGeoBuf, GeoParquet, Apache Iceberg, and OGC API-Features across data format, access patterns, scalability, and interoperability dimensions.
- Clear articulation of the complementary nature of these technologies — they serve different layers of a geospatial data stack.
- Practical decision framework for when to use each technology based on use case requirements.
