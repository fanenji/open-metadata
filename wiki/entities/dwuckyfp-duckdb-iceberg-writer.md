# dwuckyfp/duckdb-iceberg-writer

GitHub repository by [[Dwicky Feri]] containing a practical example of using DuckDB to write tables into Apache Iceberg format. The repository includes Docker Compose files for setting up a local Iceberg stack (Lakekeeper catalog and optional MinIO object store) and a SQL script (`code.sql`) that demonstrates creating, inserting, and querying an Iceberg table via DuckDB CLI.
