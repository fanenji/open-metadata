# blef.fr

Christophe Blefari's personal blog covering data, AI, and tech topics. The blog is the source of the comprehensive guide "How to manage and schedule dbt" (2022), which analyzes dbt Cloud pricing changes and provides organizational and technical guidance for managing and scheduling dbt projects in production.

## Key Contributions

- Analyzed dbt Labs' 2022 pricing changes (100% increase to $100/month/dev, 8-dev team limit, opaque Enterprise pricing).
- Provided a structured guide to dbt management covering git structures, DevEx, CI/CD, deployment, alerting, and monitoring.
- Advocated for analytics team ownership of dbt repos with data engineers focusing on DevEx.
- Recommended starting with one repo, one project for small teams.
- Documented exposures-based cross-project interfaces as an alternative to packages.
