# dbt-power-user

dbt-power-user is a VS Code extension that enhances the dbt developer experience. At [[BlaBlaCar]], it is included in the [[dev-containers-for-dbt]] configuration and provides the following features (free version):

- Run a dbt model with a single click while editing.
- Run dbt tests for that model.
- Estimate BigQuery costs for queries.
- Display the compiled SQL query.
- Preview query results.
- Generate dbt model YAML descriptions.
- Display the lineage of the dbt model.
