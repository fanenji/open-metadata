# Cal-ITP

The California Integrated Travel Project (Cal-ITP) is referenced in [[Jens Wilms]]' article as an example of dbt best practices in action. Their data-infra project uses a standardized PR template and automated reporting for comprehensive PR review. Cal-ITP serves as a model for implementing the CI/CD practices described in the article.
