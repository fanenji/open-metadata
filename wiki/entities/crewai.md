# CrewAI

CrewAI is a framework for orchestrating multi-agent AI systems where multiple agents collaborate on complex tasks. It is mentioned in the article by [[Ritam Mukherjee]] as a tool for advanced multi-agent setups in data engineering workflows.

CrewAI enables the creation of specialized agents that can work together on tasks such as pipeline orchestration, data quality monitoring, and log analysis, with each agent having defined roles and responsibilities.
