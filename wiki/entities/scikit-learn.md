# Scikit-learn

**Scikit-learn** is a robust machine learning library for Python. In the context of data engineering, it is used for advanced anomaly detection.

## Application in Data Quality
Beyond simple rule-based checks, Scikit-learn enables the implementation of statistical anomaly detection. A common technique is the use of the **Isolation Forest** algorithm to identify outliers in numerical datasets that might indicate data corruption or fraud.
