# datakitchen

datakitchen is a DataOps software platform that offers data contract capabilities. It is listed alongside [[decube]], [[dremio]], and the open-source [[datacontract.com]] as a data contract platform option in [[Jatin Solanki]]'s "Data Contracts: Implementation Guide."
