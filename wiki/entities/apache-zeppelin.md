# Apache Zeppelin

Apache Zeppelin is a web-based notebook for interactive data analytics and collaborative documentation. It provides an alternative notebook interface alongside Jupyter Notebook.

## Infrastructure Sizing

The recommended deployment is a single node with 2 vCPU, 8 GB RAM, and 50 GB storage.
