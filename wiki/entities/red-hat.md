# Red Hat

Red Hat is a technology company that published guidance on implementing robust authentication and authorization for [[model-context-protocol]] (MCP) servers. Their recommendations cover OAuth 2.1, PKCE, token validation, and RBAC integration.

## Key Findings
- Published "MCP security: Implementing robust authentication and authorization"
- Provides guidance on OAuth 2.1 with PKCE and token validation
- Recommends integration with identity providers like Keycloak
- Their guidance informs [[mcp-security-best-practices]]
