# Activity Feed

The Activity Feed is the central UI component in OpenMetadata that displays real-time updates about data assets. It serves as the primary interface for collaboration, surfacing all three pillars of [[data-collaboration|Data Collaboration]]: [[conversation-threads|Conversation Threads]], [[tasks|Tasks]], and [[announcements|Announcements]].

## Content

- **Conversation Threads:** Discussions initiated from data assets or tags.
- **Tasks:** Structured requests for descriptions, tags, or glossary term approval.
- **Announcements:** Scheduled, time-bound notifications about upcoming changes.
- **Other updates:** Changes to metadata, ownership, and other data asset properties.

## Purpose

The Activity Feed provides a unified, chronological view of all collaboration activity, enabling users to stay informed about changes and discussions relevant to their data assets. It is the primary way users discover and engage with collaboration features.
