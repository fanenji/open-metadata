# Collate AI Studio

**Collate AI Studio** is a platform provided by Collate for creating, managing, and deploying purpose-built AI agents specifically designed for data teams.

## Key Capabilities
- **Agent Management**: Define personas, assign abilities, and manage the lifecycle of data agents.
- **Ready-to-use Agents**: Provides access to specialized agents like the `DataQualityPlannerAgent`.
- **Integration**: Agents can be invoked programmatically via the [[openmetadata-ai-sdk]] using a host URL and JWT token.
- **Advanced Features**: Supports streaming, multi-turn conversations, and asynchronous execution.
