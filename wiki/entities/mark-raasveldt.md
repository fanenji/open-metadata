# Mark Raasveldt

Co-creator of DuckDB and co-author of the DuckLake manifesto. Along with Hannes Mühleisen, Raasveldt authored the manifesto that directly critiques the file-based metadata architecture of Iceberg and Delta Lake, proposing DuckLake as an alternative that uses a SQL database for all lakehouse metadata.
