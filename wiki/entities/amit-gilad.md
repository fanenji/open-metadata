# Amit Gilad

CEO and co-founder of [[LakeOps]], a company building tools to simplify and scale data infrastructure. Author of the article "Why Every Data Lake Needs a Control Plane: Lessons from Apache Iceberg," which argues that Apache Iceberg's performance gains create an operational gap requiring a centralized governance layer. Amit has a background in software engineering and system design.
