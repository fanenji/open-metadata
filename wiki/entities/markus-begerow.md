# Markus Begerow

Developer and maintainer of the [[local-llm-openmetadata-extension]], a VS Code extension that enables AI-powered data discovery by connecting to [[openmetadata]] catalogs using local or cloud LLMs.

The extension, published on the Visual Studio Marketplace, has 246 installs and supports OpenAI, [[ollama]], and custom OpenAI-compatible endpoints.
