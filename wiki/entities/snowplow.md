# Snowplow

An event streaming tool used for collecting and logging user interactions as events for analytical purposes. In the [[modern-data-stack-overview|Modern Data Stack]], Snowplow competes with [[segment]] in the event streaming space, enabling companies to track clicks, page views, and other user actions.
