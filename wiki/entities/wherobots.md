# Wherobots

A geospatial cloud platform company co-founded by [[Jia Yu]]. Wherobots hosted the livestream "Geospatial Tables in the Open Lakehouse: A New Era for Iceberg & Parquet" and developed the Havasu engine, which provides an early implementation of the Apache Iceberg GEO v3 specification. Wherobots enables users to perform ACID transactions, spatial filter pushdown, schema evolution, and time travel on geospatial data using Iceberg GEO types.
