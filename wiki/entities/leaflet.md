# Leaflet

An open-source JavaScript library for interactive web maps. Leaflet is the client-side mapping framework used by [[postholer]] to display [[cloud-optimized-geotiff-cog]] and [[flatgeobuf-fgb]] data directly from S3, enabling a fully serverless geospatial architecture. The [[georaster-layer-for-leaflet]] plugin is the critical component for COG display.
