# Tails.com dbt_artifacts Package

Community dbt package created by Tails.com for modeling dbt artifacts (run_results.json, manifest.json) into queryable tables. Referenced in [[How to add observability to your dbt deployment - Show and Tell]] as a tool that automated parts of the Snapcommerce observability system after it was initially built.
