# Daniel DuFour

Creator of [[georaster-layer-for-leaflet]], the JavaScript API that enables displaying Cloud Optimized GeoTIFFs (COGs) in Leaflet maps. His library was a critical enabler for [[scott-parks]]'s transition to a serverless geospatial architecture at [[postholer]].
