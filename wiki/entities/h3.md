# H3

**H3** is Uber's open-source hierarchical hexagonal grid system for the Earth's surface. It is a fundamental component of the [[geospatial-data-stack]] for large-scale analytics.

## Core Mechanics
- **Hierarchical Structure**: The Earth is divided into hexagons at multiple resolutions (levels 0–15).
- **Integer-Based Indexing**: H3 replaces expensive "point-in-polygon" spatial operations with high-speed "integer equality" checks.
- **Bitwise Traversal**: Parent hexagons can be computed from child hexagons using bit-shift operations, avoiding the need for expensive lookup tables.
- **Optimization**: By converting geographic points and polygons into H3 integer IDs, spatial joins in columnar warehouses (like Snowflake or DuckDB) become highly efficient integer arithmetic operations.
