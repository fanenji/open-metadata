# Spatial SQL Book (Matt Forrest)

A book by [[Matt Forrest]] published by Manning Publications, released concurrently with his Crunchy Data talk "Spatial SQL, the Modern Data Stack, and PostGIS" in December 2023. The book covers spatial SQL concepts and practical applications using PostGIS and the modern data stack.

## Details

- **Author**: Matt Forrest (Carto)
- **Publisher**: Manning Publications
- **Release**: December 2023
- **Focus**: Spatial SQL with PostGIS, integration with modern data stack tools
