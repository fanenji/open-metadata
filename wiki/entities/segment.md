# Segment

A product used for both event streaming and data integration in the [[modern-data-stack-overview|Modern Data Stack]]. Segment collects event streams (e.g., user clicks, page views) from web and mobile applications and can also be used as an [[elt-pattern|ELT]] tool to load data into cloud data warehouses. It competes with [[snowplow]] in the event streaming space.
