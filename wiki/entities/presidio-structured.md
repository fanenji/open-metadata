# Presidio Structured

Presidio Structured is a module of the [[Presidio]] framework designed for handling PII detection and anonymization in structured/tabular data (e.g., CSV files, database tables, DataFrames). It extends the capabilities of the [[Presidio Analyzer]] and [[Presidio Anonymizer]] to columnar data formats, enabling batch processing of structured datasets.
