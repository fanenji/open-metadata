# Domain-Only View

The Domain-Only View is a UI feature in [[OpenMetadata]] that enables users to navigate and operate within a specific [[domains|domain]], reducing the complexity of browsing across multiple domains.

## Key Features

- **Focused Navigation**: Users stay within a single domain for all their data needs.
- **Enhanced Discoverability**: Within a domain, users can quickly find relevant data assets and [[data-products|Data Products]].

## Relationship to Other Features

- **[[hierarchy-view]]**: The Domain-Only View may complement or be a variant of the Hierarchy View, providing a domain-scoped perspective.
- **[[persona-and-landing-page-customization]]**: The Domain-Only View may interact with Persona-based customization, potentially allowing domain-specific landing pages.

## Usage

- Encourage users to navigate and operate within their designated domains.
- Provide training and resources to help users leverage the Domain-Only View effectively.
- Regularly update domain structures to reflect organizational changes.
