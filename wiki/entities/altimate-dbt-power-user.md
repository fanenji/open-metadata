# Altimate (dbt Power User)

Altimate provides the "Power User for dbt" VS Code extension, which enhances dbt development with features like model lineage visualization, documentation generation, and query profiling. An account was created on the Altimate platform (instance name: `dpliguriadigitale`) to use the extension. The extension requires the `profiles.yaml` file to be in the project home directory.

**Security Note:** The original process document contains hardcoded credentials for the Altimate service. These should be replaced with environment variables or a secrets manager in production.
