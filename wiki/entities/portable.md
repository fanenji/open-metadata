# Portable

An [[elt-pattern|ELT]] data integration tool that focuses on the long-tail of connectors, building a wide repository of integrations for less common data sources. In the [[modern-data-stack-overview|Modern Data Stack]], Portable competes with [[fivetran]] and [[airbyte]] in the data integration layer.
