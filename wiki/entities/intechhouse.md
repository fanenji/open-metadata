# InTechHouse

InTechHouse is a consulting firm that proposed a maturity model for [[data-mesh]] with explicit performance baselines. Their framework evaluates maturity across:

- **Accessibility** — How easily data can be discovered and accessed
- **Quality** — Data accuracy, freshness, and completeness
- **Productivity** — Time-to-insight and developer velocity
- **Compliance** — Governance and security adherence

The model defines progressive stages against industry best practices and is one of several frameworks synthesized in the [[data-mesh-maturity-assessment]].
