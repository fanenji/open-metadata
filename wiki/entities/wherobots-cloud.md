# Wherobots Cloud

**Wherobots Cloud** is a managed, cloud-based notebook environment optimized for geospatial analysis and workloads. It provides a streamlined, end-to-end workflow for processing large-scale spatial datasets and executing [[apache-sedona]] queries through a seamless interface.

## Key Features

- **Managed Notebooks**: A Python and SQL-based environment pre-configured with the geospatial Python ecosystem (e.g., [[geopandas]], [[matplotlib]], and `SedonaPyDeck`).
- **File Management**: An integrated interface for uploading and managing spatial datasets (e.g., Shapefiles, CSVs) via S3-compatible storage.
- **Integrated Visualization**: Support for both interactive (via `SedonaPyDeck`) and static (via `Matplotlib`) geospatial visualizations directly within the notebook.
- **Distributed Computing**: Seamless integration with [[apache-sedona]] for distributed spatial computing.
