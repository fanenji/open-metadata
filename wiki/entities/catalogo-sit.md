# Catalogo SIT

The Catalogo SIT is a configuration source within the SDI of Regione Liguria that defines layer definitions. It is used by the Geoscript system to read the configuration of layers and construct the appropriate configuration for ogr2ogr commands during the Oracle-to-PostGIS data copy phase.
