# Apache Samza

Apache Samza is a stream processing framework built on top of Apache Kafka. It was developed at LinkedIn, where Martin Kleppmann worked on it after the Rapportive acquisition.

## Relationship to Kafka

Samza processes data streams from Kafka, providing a framework for building real-time stream processing applications. It was one of the early stream processing systems that demonstrated the power of the Kafka log abstraction.
