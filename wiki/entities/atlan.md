# Atlan

A data cataloging, documentation, discovery, and access control platform used in the [[modern-data-stack-overview|Modern Data Stack]]. Atlan helps organizations document and discover data assets and manage data privacy and access controls. It competes with open-source solutions like [[amundsen]], [[datahub]], and [[metacat]].
