# Dr. James Williams

Author of the tutorial "Getting Started with H3: The Hexagonal Grid System for Spatial Analysis," which provides an introduction to Uber's H3 hexagonal grid system. The tutorial covers H3 fundamentals, resolution levels, and practical usage patterns. Dr. Williams' work is cited in the [[hex-egg-table]] synthesis as a foundational reference for H3 concepts.
