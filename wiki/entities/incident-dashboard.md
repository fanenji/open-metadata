# Incident Dashboard

The **Incident Dashboard** is the central UI hub in OpenMetadata where all data quality incidents are displayed. It provides filtering capabilities to help users manage and prioritize incidents effectively.

## Filters Available

- **Assignee** — View incidents assigned to specific team members.
- **Status** — Filter incidents based on their current status (New, ACK, Assigned, Resolved).
- **Test Cases** — Filter incidents associated with specific test cases.
- **Time** — Sort incidents by the time they were reported or last updated.

## Usage

The Incident Dashboard is the entry point for the [[incident-manager]] workflow. From the dashboard, users can select an incident to change its status, view test case details, or navigate to the incident's activity timeline.
