# Pinterest Engineering

The engineering organization at Pinterest, responsible for building and maintaining the company's data infrastructure and analytical tools. They developed and deployed the Text-to-SQL feature described in the blog post "How we built Text-to-SQL at Pinterest."

Key contributors to the Text-to-SQL project include Adam Obeng (Data Scientist), J.C. Zhong (Tech Lead), and Charlie Gu (Sr. Manager, Engineering).

Pinterest Engineering maintains [[Querybook]], an open-source big data SQL query tool, which served as the deployment platform for the Text-to-SQL feature.
