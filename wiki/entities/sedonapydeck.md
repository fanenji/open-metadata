# SedonaPyDeck

[[sedonapydeck]] is a Python package designed to facilitate the creation of interactive geospatial visualizations (specifically choropleth maps) directly within the [[apache-sedona]] ecosystem.

It allows users to:
- Generate interactive maps within notebook environments like [[wherobots-cloud]].
- Easily create choropleth layers from spatial DataFrames.
- Bridge the gap between distributed spatial computing and modern web-based visualization tools like Deck.gl.
