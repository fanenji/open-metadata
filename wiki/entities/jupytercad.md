# JupyterCAD

JupyterCAD is a collaborative CAD modeler built as a JupyterLab extension. It is a sibling project to [[JupyterGIS]], sharing the same architectural approach based on the [[Yjs framework]] for CRDT-based collaborative editing.

JupyterCAD serves as a precedent for JupyterGIS, demonstrating the feasibility of building collaborative authoring tools on the JupyterLab application framework. The notebook integration architecture developed for JupyterCAD will be reused for JupyterGIS.
