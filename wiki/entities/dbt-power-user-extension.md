# dbt Power User Extension

A VS Code extension chosen by [[BlaBlaCar]] as the new SQL development environment for their dbt migration. Replaced the BigQuery UI as the primary tool for writing and testing SQL queries. Part of the [[dbt-developer-experience]] strategy to address the loss of the familiar BigQuery UI workflow.
