# Development Seed

Development Seed is a geospatial technology company that builds open-source tools and infrastructure for processing and visualizing geospatial data, with a focus on earth observation, mapping, and spatial data analysis. They are contributors to the [[GeoArrow]] and [[GeoParquet]] ecosystems.

## Key Projects

- **[[lonboard]]** — Python-to-deck.gl bridge using GeoArrow/GeoParquet
- **[[geoarrow-layers]]** — JavaScript library for deck.gl binary API with GeoArrow
- **[[geoarrow-rust]]** — Rust implementation of the GeoArrow specification
- **WASM GeoParquet parser** — Rust-based WebAssembly bindings for browser-side GeoParquet parsing
