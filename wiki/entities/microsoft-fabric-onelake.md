# Microsoft Fabric OneLake

Microsoft's unified data lake platform that supports Apache Iceberg. The Microsoft and Snowflake partnership announcement frames Iceberg as a foundation for bidirectional access between Snowflake and Fabric. This is cited as a key signal of vendor standardization around Iceberg.
