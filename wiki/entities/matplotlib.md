# Matplotlib

[[matplotlib]] is a comprehensive library for creating static, animated, and interactive visualizations in Python. In the context of geospatial workflows, it is used to render thematic maps such as choropleths.

Key uses in geospatial analysis:
- Plotting color-coded polygons (choropleths) based on statistical variables.
- Applying color ramps (e.g., 'YlGn', 'RdYlGn') to represent data distributions.
- Adding annotations, legends, and titles to spatial plots.
- Saving high-resolution maps for reports.
