# Planet Labs

A satellite imagery company where [[Chris Holmes]] works as a Planet Fellow. Planet Labs provides daily satellite imagery of the Earth's surface and is interested in using Iceberg as a standard format for distributing geospatial data products, enabling customers to access data across multiple cloud platforms without vendor lock-in.
