# Personal Access Token (PAT)

A **Personal Access Token (PAT)** is a secure credential used to authenticate programmatic access to Dremio. In the context of the [[model-context-protocol]], the PAT is required to authorize the [[dremio-mcp-server]] to interact with the Dremio engine on behalf of the user or agent.
