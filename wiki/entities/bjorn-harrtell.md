# Björn Harrtell

Creator of [[flatgeobuf-fgb]], the indexed vector data format that uses HTTP range-request streaming for cloud-native geospatial data delivery. His work was instrumental in enabling [[scott-parks]]'s serverless geospatial architecture at [[postholer]].
