# Will Sargent

Will Sargent is a Senior Resident Architect at dbt Labs. He is the instructor for the "Advanced dbt Cloud Setup Workshop" course, which is part of the [[dbt-cloud-architect-certification]] learning path (Milestone #5: Exam Preparation). The workshop covers the 7 key milestones for dbt Cloud setup for enterprise, including platform configuration, project setup, user permissions, environment management, version control, SSO integration, job scheduling, and troubleshooting.
