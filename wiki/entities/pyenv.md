# PyEnv

**PyEnv** is a Python version management tool that allows users to easily install, manage, and switch between multiple versions of Python on a single system.

### Key Features
- **Isolation:** Prevents interference with the system's default Python installation.
- **Granularity:** Enables setting Python versions globally, per-shell, or per-project (using `.python-version` files).
- **Simplicity:** Provides a lightweight alternative to heavier managers like Conda.

It is often used as the foundation for more complex environment management tools like [[poetry]].
