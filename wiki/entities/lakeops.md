# LakeOps

A platform that provides a centralized control plane for data lake governance, orchestrating table maintenance policies across catalogs, query engines, and teams. LakeOps enforces consistent retention policies, tracks snapshot tag lifecycles, and coordinates maintenance procedures like snapshot expiration, compaction, and orphan file cleanup. The platform is presented as the solution to the operational gap created by Apache Iceberg's decentralized maintenance model. Founded by [[Amit Gilad]].
