# fabrice-etanchaud

GitHub user and maintainer of the [dbt-dremio](https://github.com/fabrice-etanchaud/dbt-dremio) adapter, which enables dbt to transform and test data within Dremio. The adapter is a key integration point for using dbt with the Dremio data lakehouse engine.
