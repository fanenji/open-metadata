# EPSG:7791

EPSG:7791 (RDN2008 / UTM zone 32N) is the target coordinate reference system for vector data transformation during the PostGIS-to-DP ingestion phase. Data read from PostGIS is transformed to this CRS before being written to the DP's object storage in GeoParquet/Iceberg format.
