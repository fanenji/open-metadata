# djouallah

GitHub user and creator of the [[semantic-sql-testing-benchmark]], a benchmark for evaluating Small Language Models (SLMs) on SQL generation tasks using a semantic model and exact result matching. The benchmark tests models like Qwen3.5-4B, Qwen3-4B-Instruct-2507, and Gemma4-E4B on a TPC-DS star schema.
