# Kraina AI

Kraina AI is the organization that maintains and develops [[quackosm|QuackOSM]], an open-source tool for reading OpenStreetMap PBF files using DuckDB. The organization focuses on geospatial data processing tools that leverage modern cloud-native architectures.
