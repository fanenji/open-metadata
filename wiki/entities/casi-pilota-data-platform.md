# Casi Pilota Data Platform

Two pilot use cases identified in the Data Platform integration plan to validate the integration middleware before full-scale rollout:

1. **RQA (pilota):** A pilot use case for validating integration patterns. The specific domain and requirements of RQA are not detailed in the planning document.

2. **Geo Data Ingestion (pilota):** A pilot use case for validating geospatial data ingestion into the Data Platform. This connects to the existing [[geospatial-etl-pipeline-iceberg]] work stream.

These pilots are intended to de-risk the integration middleware development by testing real-world scenarios early.
