# Datafold

Datafold is a company that provides data observability and data diff tools for data teams. Their blog published the article "What is CI, and why should you care about it" by [[Gleb Mezhanskiy]]. Datafold's primary product is a data diff tool that compares dev and prod versions of tables to show the impact of code changes. This tool is promoted as a "bonus" step in the [[dbt-ci-pipeline-anatomy]].
