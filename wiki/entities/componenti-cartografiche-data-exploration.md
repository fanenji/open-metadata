# Componenti Cartografiche per Data Exploration

A planned set of map-based visualization components for the Data Platform's exploration interface. These components are critical for geospatial data use cases, enabling users to interact with spatial data through cartographic displays. The specific technology (e.g., MapLibre, Leaflet, Deck.gl) and integration patterns with the platform's virtualization layer ([[dremio]]) and exploration tools ([[duckdb]], [[marimo]]) are not yet specified.
