# Sifflet

A Full Data Stack Observability platform, founded by [[Salma Bakouk]]. Sifflet provides an overseeing layer for the modern data stack, enabling organizations to gain automated data reliability at every step of the pipeline — from ingestion through storage and processing to BI tools. The platform leverages the three pillars of data observability: metrics, metadata, and lineage.
