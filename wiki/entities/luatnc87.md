# luatnc87

GitHub user and author of the repository "modern-data-warehouse-modeling-and-data-quality-with-dbt-openmetadata." The repository provides a comprehensive, runnable guide to building a modern data warehouse using DuckDB, dbt, OpenMetadata, and Apache Airflow, with a [[hybrid-dimensional-obt-modeling]] approach and a [[custom-connector-openmetadata|custom DuckDB connector for OpenMetadata]].

## Key Contributions

- **Hybrid Modeling Pattern**: Documented the two-layer dimensional + OBT architecture.
- **Custom DuckDB Connector**: Built and shared a working DuckDB connector for OpenMetadata.
- **dbt DAG Generator**: Demonstrated parsing `manifest.json` to create Airflow DAGs.
- **End-to-End Integration**: Provided a complete Docker Compose setup for the open-source stack.
