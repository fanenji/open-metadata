# GoCardless

GoCardless is a payment company where [[Andrew Jones]] implemented the [[data-contract]] concept. This implementation is a frequently cited reference in the data contracts community and is acknowledged by [[Jatin Solanki]] in the "Data Contracts: Implementation Guide."
