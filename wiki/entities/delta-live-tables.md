# Delta Live Tables

[[databricks|Databricks']] data transformation framework that enables declarative pipeline development. In the [[modern-data-stack-overview|Modern Data Stack]], Delta Live Tables is an alternative to [[dbt]] for defining transformations, alongside other options like [[lookml]] and [[apache-airflow]].
