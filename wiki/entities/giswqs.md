# giswqs (Qiusheng Wu)

**Qiusheng Wu** is a geospatial educator and open-source contributor, known for creating and maintaining the [[geog-414-spatial-data-management]] course repository at the University of Tennessee. He is also the author of the `geemap` Python package, which provides interactive mapping and analysis capabilities for Google Earth Engine.

His GitHub profile (`giswqs`) hosts numerous geospatial learning resources, tutorials, and tools that emphasize modern, cloud-native approaches to spatial data management. The [[geog-414-spatial-data-management]] course is a key example of the [[course-as-code]] pattern, delivering a full university curriculum as a public, version-controlled repository.
