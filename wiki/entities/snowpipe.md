# Snowpipe

[[snowflake|Snowflake's]] streaming data integration service that enables loading data in near real-time. In the [[modern-data-stack-overview|Modern Data Stack]], Snowpipe (along with [[confluent|Confluent/Apache Kafka]]) enables streaming data directly into the cloud warehouse for real-time use cases.
