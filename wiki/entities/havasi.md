# Havasu

Wherobots' cloud query engine that provides an early implementation of the Apache Iceberg GEO v3 specification. Havasu supports ACID transactions, spatial filter pushdown, schema evolution, and time travel on geospatial data using native Iceberg GEO types. It serves as a reference implementation for the community while other engines are developing their support.
