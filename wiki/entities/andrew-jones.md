# Andrew Jones

Andrew Jones is credited as the creator of the [[data-contract]] concept. He implemented data contracts at [[GoCardless]], a reference implementation that is frequently cited in the data contracts community. His work is acknowledged by [[Jatin Solanki]] in the "Data Contracts: Implementation Guide."
