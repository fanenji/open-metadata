# WhyLabs

WhyLabs is an AI observability platform that provides monitoring and debugging capabilities for machine learning models and data pipelines. It is mentioned in the article by [[Ritam Mukherjee]] as an observability add-on that can be integrated with LLM plugins for smart monitoring in data engineering workflows.

WhyLabs helps detect data drift, model degradation, and pipeline anomalies, complementing the AI-assisted log analysis approach described in the article.
