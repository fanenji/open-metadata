# uv

**uv** is an extremely fast Python package installer and resolver, written in Rust. It is used as the primary package manager for running and managing the [[dremio-mcp-server]] and other modern Python-based data tools.

In the context of the Data Platform, `uv` ensures reproducible environments and rapid deployment of agentic tools and MCP servers.
