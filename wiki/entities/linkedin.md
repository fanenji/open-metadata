# LinkedIn

LinkedIn is a professional social network and a significant contributor to open-source data infrastructure. It was the birthplace of several foundational data systems technologies.

## Key Contributions

- **Kafka**: Distributed streaming platform, created by Jay Kreps.
- **Samza**: Stream processing framework on top of Kafka.
- **Voldemort**: Distributed key-value store.
- **Data Infrastructure**: LinkedIn's large-scale Hadoop cluster and stream processing systems provided the environment where Martin Kleppmann and others gained hands-on experience with distributed systems at scale.

## Relevance to Data Platform

LinkedIn's data integration challenges (moving data between many sources and sinks) mirror the core problems that modern data platforms solve. The technologies born at LinkedIn (especially Kafka) are now industry standards.
