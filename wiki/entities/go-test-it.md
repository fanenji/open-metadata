# Go Test It

Go Test It was Martin Kleppmann's first startup (circa 2008). It was a cross-browser automated testing service for websites, based on the open-source Selenium project.

## Product

The service allowed developers to write test scripts that automated user interactions with a website and checked for correct behavior across different browsers. It was provided as a hosted service so customers didn't need to run their own VMs with various operating systems.

## Outcome

The product worked technically but struggled with adoption. Developers found it difficult to integrate into their workflows and build the habit of writing test scripts. The startup did not achieve significant revenue or traction.
