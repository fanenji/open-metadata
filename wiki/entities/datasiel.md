# Datasiel

Datasiel is the organization that hosts the reference GeoServer instance used in the [[api-configurator-system]] documentation. The example URLs point to a GeoServer instance at `geoservizi.datasiel.net:8083`.

## Role

- Hosts the reference GeoServer OGC API - Features implementation
- Provides infrastructure for the legacy geospatial data serving layer
