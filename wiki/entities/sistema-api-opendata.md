# Sistema API OpenData

A planned subsystem of the Data Platform that provides a programmatic API layer for serving open data to external consumers. This subsystem enables machine-to-machine access to platform data, complementing the [[data-download-service]] (Dynamic Download) which focuses on synchronous file-based delivery. The API design, protocol, and authentication mechanisms are not yet specified in the planning document.
