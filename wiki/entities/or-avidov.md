# Or Avidov

Author of the article "dbt observability 101: How to monitor dbt run and test results" and likely a founder or key contributor to the [[elementary-dbt-package]] open-source project. The article presents a practical, dbt-native approach to monitoring dbt execution metadata using Jinja variables and `on-run-end` hooks.

## Contributions

- Popularized the concept of **dbt observability** as a distinct practice within the dbt ecosystem.
- Provided a step-by-step implementation guide that avoids the complexity of processing dbt artifact JSON files.
- Developed the [[elementary-dbt-package]], which extends the basic pattern with richer metadata collection, dashboards, and Slack alerts.
