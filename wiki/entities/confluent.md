# Confluent

**Confluent** is the managed version of [[apache-kafka|Apache Kafka]], a streaming data integration platform offered by Confluent, Inc. (a streaming platform vendor). In the [[modern-data-stack-overview|Modern Data Stack]], Confluent—along with [[snowpipe]]—enables streaming data directly into cloud data warehouses for real-time use cases. Additionally, Confluent published a resource titled "Data Mesh - 7 Effective Practices to Get Started," which is referenced in the [[data-mesh-maturity-assessment]] as a source for practical implementation advice.
