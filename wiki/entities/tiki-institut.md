# TIKI-Institut

An organization that developed and maintains the custom [[openmetadata-dremio-connector]], enabling metadata ingestion from [[dremio]] into [[openmetadata]].

## Role

TIKI-Institut created the connector to address the lack of native Dremio support in OpenMetadata. The connector is hosted on GitHub under the [TIKI-Institut](https://github.com/TIKI-Institut) organization.
